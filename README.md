# Flask Інтернет-Магазин

Простий інтернет-магазин на Flask з адмін-панеллю для управління товарами та замовленнями.

## Функціонал

### Публічна частина:
- Каталог товарів з пошуком та фільтрацією по категоріям
- Детальна сторінка товару
- Реєстрація та авторизація користувачів
- Кошик (додавання, редагування кількості, видалення)
- Оформлення замовлення
- Історія замовлень користувача

### Адмін-панель:
- Управління товарами (CRUD операції)
- Управління замовленнями (перегляд, зміна статусу)
- Статистика (загальна кількість товарів, замовлень, користувачів)
- Топ товарів за продажами

## Встановлення

1. Встановіть залежності:
```bash
pip install -r requirements.txt
```

2. Запустіть додаток:
```bash
python app.py
```

3. Відкрийте браузер та перейдіть за адресою: `http://localhost:5000`

## Технології

- Flask 3.0.0

- Requests (для Telegram інтеграції)

## Telegram інтеграція

Для отримання повідомлень про нові замовлення в Telegram групу:

1. Створіть бота через [@BotFather](https://t.me/BotFather) в Telegram
2. Отримайте токен бота
3. Додайте бота в групу і отримайте chat_id групи (можна використати [@userinfobot](https://t.me/userinfobot) або [@getidsbot](https://t.me/getidsbot))
4. Встановіть змінні оточення:
   ```bash
   export TELEGRAM_BOT_TOKEN="ваш_токен_бота"
   export TELEGRAM_CHAT_ID="ваш_chat_id_групи"
   export TELEGRAM_ENABLED="true"
   ```

Або встановіть їх безпосередньо в `config.py`.

**Примітка:** Якщо `TELEGRAM_ENABLED` не встановлено або встановлено в `false`, повідомлення не будуть відправлятися.

## Примітки

- Товари не мають цін (без оплати)
- База даних SQLite (shop.db) створюється автоматично
- Всі паролі хешуються через Werkzeug
- Адмін-панель доступна тільки для користувачів з is_admin=True
